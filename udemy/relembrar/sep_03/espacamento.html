Lógica dos Espaçamentos em Design Web

Entenda quando e como usar cada tipo de espaçamento em seus projetos

Sistema de Espaçamentos

Um bom sistema de espaçamentos é baseado em uma unidade modular (geralmente 4px, 8px ou 12px) da qual todos os outros espaçamentos derivam.

Escala de Espaçamentos (base: 8px)

xxs 4px
xs 8px
sm 16px
md 24px
lg 32px
xl 40px
xxl 48px
xxxl 64px
Dica: Usar uma escala baseada em múltiplos de 4px ou 8px ajuda a criar consistência visual e alinhamento preciso em todos os elementos da interface.

Quando Usar Cada Espaçamento

4px (xxs)

Espaçamentos mínimos entre elementos muito próximos, ajustes finos de alinhamento.

8px (xs)

Espaçamento interno de componentes pequenos, margens entre elementos relacionados.

16px (sm)

Espaçamento padrão entre elementos, padding interno de componentes, margens entre seções relacionadas.

24px (md)

Espaçamento entre grupos de elementos, margens médias, padding de cartões.

32px (lg)

Espaçamento entre seções distintas, margens maiores, padding de containers.

40px (xl)

Espaçamento entre componentes maiores, seções de página, margens laterais de layout.

48px (xxl)

Espaçamento entre elementos grandes, seções principais de página.

64px (xxxl)

Espaçamentos grandes entre seções principais, margens superiores/inferiores de páginas.

Boas Práticas de Espaçamento

Consistência é fundamental: Use o mesmo sistema de espaçamento em todo o projeto.
Hierarquia visual: Use espaçamentos maiores para separar grupos de conteúdo diferentes e menores para elementos relacionados.
Responsividade: Ajuste os espaçamentos para diferentes tamanhos de tela. Em dispositivos móveis, espaçamentos muito grandes podem desperdiçar espaço valioso.
Use variáveis CSS: Defina seus espaçamentos como variáveis CSS para facilitar a manutenção e consistência.
/* Defina suas variáveis de espaçamento */
:root {
  --space-unit: 8px;
  --space-xs: var(--space-unit);
  --space-sm: calc(var(--space-unit) * 2);
  --space-md: calc(var(--space-unit) * 3);
  --space-lg: calc(var(--space-unit) * 4);
  --space-xl: calc(var(--space-unit) * 5);
}

/* Use as variáveis em seu CSS */
.card {
  padding: var(--space-md);
  margin-bottom: var(--space-lg);
}
