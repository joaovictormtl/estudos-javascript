Padronização de Títulos e Parágrafos

Um sistema tipográfico escalável com proporções harmoniosas

Exemplo de Uso

Título Principal (h1)

Este é um parágrafo de destaque (lead) que introduz o conteúdo abaixo.

Este é um parágrafo comum. Ele contém texto de exemplo para demonstrar o tamanho, altura de linha e espaçamento padrão para conteúdo corporal.

Subtítulo (h2)

Outro parágrafo comum que segue um subtítulo. Observe o espaçamento consistente entre os elementos.

Título de Seção (h3)

Texto de parágrafo normal abaixo de um título de seção. A hierarquia visual é mantida através do tamanho e peso da fonte.

Título de Subseção (h4)

Parágrafo seguindo um título de subseção. Mesmo com múltiplos níveis, a consistência é mantida.

Título menor (h5)

Este parágrafo demonstra o texto abaixo de um título de quinto nível, que é frequentemente usado para agrupamentos menores.

Título mínimo (h6)

O nível mais baixo de título, geralmente usado para elementos muito específicos ou notas.

Este é um texto pequeno, ideal para notas de rodapé, legendas ou informações secundárias.

Escala Tipográfica

Nossa escala usa uma proporção de 1.250 (Major Third) para criar harmonia visual entre os tamanhos:

h1 Título Principal var(--text-xxxl)
h2 Subtítulo var(--text-xxl)
h3 Título de Seção var(--text-xl)
h4 Título de Subseção var(--text-lg)
h5 Título Menor var(--text-md)
h6 Título Mínimo var(--text-sm)
Parágrafo Texto corporal padrão var(--text-sm)
Lead Parágrafo de destaque var(--text-md)
Pequeno Texto secundário pequeno var(--text-xs)

Código CSS

:root {
  --text-base-size: 1rem;
  --text-scale-ratio: 1.250;
  
  --text-xs: calc(var(--text-base-size) / var(--text-scale-ratio));
  --text-sm: calc(var(--text-base-size));
  --text-md: calc(var(--text-sm) * var(--text-scale-ratio));
  --text-lg: calc(var(--text-md) * var(--text-scale-ratio));
  --text-xl: calc(var(--text-lg) * var(--text-scale-ratio));
  --text-xxl: calc(var(--text-xl) * var(--text-scale-ratio));
  --text-xxxl: calc(var(--text-xxl) * var(--text-scale-ratio));
}

h1 { font-size: var(--text-xxxl); font-weight: 800; }
h2 { font-size: var(--text-xxl); font-weight: 700; }
h3 { font-size: var(--text-xl); font-weight: 600; }
h4 { font-size: var(--text-lg); font-weight: 600; }
h5 { font-size: var(--text-md); font-weight: 600; }
h6 { font-size: var(--text-sm); font-weight: 600; }

p { font-size: var(--text-sm); }
.lead { font-size: var(--text-md); }
.small { font-size: var(--text-xs); }
